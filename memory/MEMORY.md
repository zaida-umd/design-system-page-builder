# Memory Index

## Feedback
- [Test pages location](feedback_test_pages_location.md) — Always write test pages to the main repo's `/test/` directory, never inside `.claude/`
- [Card aligned default](feedback_card_aligned_default.md) — Always use `data-aligned` on `umd-element-card` in grids so images crop to consistent height
- [Layout variety over straight grids](feedback_layout_variety.md) — Prefer masonry, overlay cards, or pathways over repetitive straight card grids
- [Center hero text when followed by non-full-width content](feedback_hero_centered_text.md) — Add data-layout-text="center" when hero is followed by quote, masonry, or constrained-width sections
